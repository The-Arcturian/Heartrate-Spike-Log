**Spike Tracker**

A lightweight, mobile-friendly heart rate spike logger built as a single HTML file. No account, no server, no installation — open it in your browser, add it to your home screen, and it runs like an app.

Log spikes with a few taps, tag contributing variables, and track patterns over time. All data stays on your device in browser localStorage. Use the built-in JSON export to back up your data or move it between devices.

Built for personal use. Free to use and fork.
